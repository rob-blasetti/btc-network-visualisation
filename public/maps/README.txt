Place optional map textures here to improve visual fidelity.

Supported filenames (the app will try in this order):

Base map (land/sea):
- world-dark-4096.jpg
- world-dark-2048.jpg
- world-dark-2048.png

Boundaries/Coastlines overlay (transparent):
- world-admin0-boundaries-4096.png
- world-admin0-boundaries-2048.png
- world-coastlines-2048.png

Recommended public-domain sources:
- NASA Blue Marble (public domain): https://visibleearth.nasa.gov/collection/1484/blue-marble
- Natural Earth raster (public domain): https://www.naturalearthdata.com/downloads/10m-raster-data/10m-natural-earth-2/
- Wikimedia “Blank world map” (check license):
  Dark gray example:
  https://upload.wikimedia.org/wikipedia/commons/thumb/2/29/BlankMap-World-v2-dark-gray.svg/2048px-BlankMap-World-v2-dark-gray.svg.png

How to use:
1) Download 2048–4096px images.
2) Save them with one of the supported filenames above in this folder.
3) Commit and push; GitHub Pages will redeploy and the app will auto-pick them up.
